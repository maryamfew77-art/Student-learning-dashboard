Student Learning Dashboard

## Features

- Responsive and modern student learning dashboard interface.
- Live digital clock and current date display.
- Add new courses with details such as course name, duration, difficulty level, and status.
- Dynamic course cards generated using JavaScript constructor functions and DOM manipulation.
- Real-time statistics showing total, ongoing, and completed courses.
- Course search functionality for quickly finding specific subjects.
- Clean sidebar navigation with sections for courses, progress, notes, and settings.
- Fully responsive layout built with HTML, CSS Grid, and Flexbox for desktop and mobile devices.

This project was developed using **HTML5**, **CSS3**, and **Vanilla JavaScript** to practice object-oriented programming, dynamic rendering, and responsive web design.
